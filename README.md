# 🛒 E-Commerce Dashboard (Power BI)

## 📊 Overview

This project presents an interactive E-commerce dashboard built using Microsoft Power BI. The dashboard provides insights into sales performance, customer behavior, and overall business trends.

## 🎯 Objectives

* Analyze overall sales and revenue trends
* Understand customer purchasing behavior
* Identify top-performing products and categories
* Track key performance indicators (KPIs)

## 📌 Key Features

* Sales overview (Total Revenue, Orders, Profit)
* Category-wise and product-wise analysis
* Customer segmentation insights
* Time-based trends (daily, monthly, yearly)
* Interactive filters and slicers
